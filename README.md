📖 Wordle - 7 Days Of Code (Alura)
Este é o resultado do meu desafio #7DaysOfCode da Alura! Durante uma semana, desenvolvi do zero a minha própria versão do famoso jogo Wordle (conhecido no Brasil como "Termo").

O objetivo é adivinhar uma palavra secreta de 5 letras em até 6 tentativas. O jogo dá dicas através de cores:

🟩 Verde: A letra faz parte da palavra e está na posição correta.

🟨 Amarelo: A letra faz parte da palavra, mas está na posição errada.

⬛ Cinza: A letra não faz parte da palavra.

🚀 Como Jogar
Você pode testar o jogo agora mesmo clicando no link abaixo:
👉 [(https://kauan-godoi.github.io/wordle-game/)]

🛠️ Tecnologias Utilizadas
HTML5 para a estrutura do tabuleiro e teclado virtual.

CSS3 para o layout (estilo Dark Mode) e responsividade para dispositivos móveis.

JavaScript (ES6+) para toda a lógica do jogo e manipulação do DOM.

Toastify JS para as notificações flutuantes na tela.

🧠 O que eu aprendi
Neste desafio, foquei em evoluir minhas habilidades técnicas e de organização:

Manipulação de Arrays: Filtragem e comparação de letras entre o palpite e a palavra sorteada.

Eventos de Teclado: Captura de teclas físicas e cliques no teclado virtual.

Consumo de JSON: Busca da lista de palavras de um banco de dados externo (database.json) usando fetch.

Clean Code (Código Limpo): Apliquei boas práticas para deixar o código mais legível e organizado, utilizando funções com responsabilidades únicas e nomes de variáveis claros.

Deploy: Configuração e publicação do projeto no GitHub Pages.
